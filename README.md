Collection of personal typescript utils.

## Installation

```shell script
npm i --save @ambroise-rabier/ts-utils
```

To have extensions available, add this line in your entrypoint (`main.ts`, `index.ts` usually).
```typescript
import '@ambroise-rabier/ts-utils';
```
